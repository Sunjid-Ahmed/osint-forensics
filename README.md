# 🛰️ OSINT & Forensics — Ultimate Learning Platform

An advanced, interactive learning platform for **OSINT (Open Source Intelligence)** and **Digital Forensics**.

Designed as a visual field manual, this project combines structured learning paths, tools, workflows, and real-world techniques into a single, immersive interface.

---

## 🚀 Features

* 🗺️ **Structured Learning Roadmaps**

  * OSINT progression (Beginner → Elite)
  * Digital Forensics workflow

* 🔍 **OSINT Modules**

  * Passive Recon
  * Active OSINT
  * People Intelligence
  * Infrastructure Intelligence

* 🧠 **Digital Forensics**

  * Evidence acquisition
  * Memory forensics
  * Windows artifacts
  * Network forensics

* 🧪 **Advanced Topics**

  * Malware analysis
  * CTF workflows
  * OPSEC strategies

* 🛠️ **Tool Mastery Section**

  * Popular OSINT & DF tools
  * Practical usage references

* 📚 **Resources & Writeups**

  * Curated learning materials
  * Real-world case insights

* 🎨 **Modern UI**

  * Cyberpunk-style interface
  * Animated visualizations
  * Interactive navigation system
